4. OWL Reasoning Rules — Auto-Inferred Classes

ApprovedTreatment
IF a Drug
AND treats some Disease
AND approvalStatus = "Approved"
→ THEN: ApprovedTreatment
Immunotherapy
IF a Drug
AND targets some ImmuneCheckpointProtein
→ THEN: Immunotherapy
HighImpactResearcher
IF a Researcher
AND hIndex ≥ 70
→ THEN: HighImpactResearcher
DefinitiveEvidence
IF a ClinicalTrial
AND phase = "Phase 3"
AND trialStatus = "Completed"
→ THEN: DefinitiveEvidence
EpidemicDisease
IF a Disease
AND prevalence = "Very High"
→ THEN: EpidemicDisease
